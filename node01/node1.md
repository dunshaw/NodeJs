# node
## what is node ?
    + chrome V8 runtime
    + 事件驱动
    + 非阻塞的i/O
    i/o ：input output 输入输出流，正常下i/o的操作都是阻塞的（ajax同步）
    网络请求 数据库处理 文件的读写 。。。。都是阻塞的

    优点：高并发特别好（仅次于pyhton）


## why need node ？
    1.防止甩锅（明确数据交互错误来源）
    2.能够书写api 斜杠青年
    3.了解前后端的交互流程
    4.全栈（全干）工程
